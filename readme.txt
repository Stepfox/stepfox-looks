Stepfox Looks is a companion plugin that enhances the WordPress block editor with advanced responsive controls, custom blocks, and editor extensions. It adds professional design tools and UX features to help you build modern magazine and news sites faster.

== Description ==

Stepfox Looks is a powerful companion plugin that extends the WordPress block editor with advanced functionality and responsive controls. This plugin is designed to work seamlessly with Stepfox themes, providing professional-grade customization options.

= Key Features =

* **Metafield Block**: Display custom fields and metadata with advanced rendering options
* **Load More Block**: AJAX-powered pagination for query blocks
* **Responsive Extensions**: Comprehensive responsive controls for all blocks
* **Social Share Extensions**: Enhanced social sharing functionality
* **Cover Block Extensions**: Advanced cover block customization
* **Post Template Fallback**: Intelligent fallback system for post templates

= Custom Blocks =

* **Metafield Block**: Render custom fields with multiple display options (text, image, link, button)
* **Load More Block**: Add AJAX load more functionality to query blocks

= Block Extensions =

* **Responsive Controls**: Add responsive settings to all blocks
* **Social Share**: Enhanced social link blocks with sharing functionality
* **Cover Block**: Extended cover block options
* **Post Template Fallback**: Prevents template breaking when switching post types

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/stepfox-looks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin will automatically extend the block editor with new functionality.

== Requirements ==

* WordPress 6.0 or higher
* PHP 7.4 or higher
* Stepfox theme (recommended for optimal compatibility)

== Frequently Asked Questions ==

= Does this plugin work with any theme? =

While the plugin will function with most themes, it's specifically designed to work optimally with Stepfox themes.

= Can I use the responsive controls with any block? =

Yes, the responsive extensions add responsive controls to all registered blocks in the block editor.

= Is the metafield block compatible with ACF (Advanced Custom Fields)? =

Yes, the metafield block can display ACF fields along with standard WordPress custom fields.
